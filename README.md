# Бобер кликер

Игра-кликер "Бобер кликер" - увлекательная игра, где вы кликаете по бобру, чтобы зарабатывать очки и покупать улучшения.

## Структура проекта

```
├── index.html              # Главная страница
├── README.md              # Этот файл
├── assets/                # Ресурсы
│   ├── css/               # Стили
│   │   └── style.css      # Основные стили
│   ├── js/                # JavaScript файлы
│   │   ├── main.js        # Основной скрипт
│   │   ├── auth.js        # Авторизация
│   │   └── game.js        # Логика игры
│   ├── img/               # Изображения
│   │   ├── skins/         # Скины бобра
│   │   └── icons/         # Иконки
│   └── sounds/            # Звуковые файлы
├── api/                   # Серверные скрипты
│   ├── login.php          # Вход
│   ├── register.php       # Регистрация
│   ├── save-score.php     # Сохранение очков
│   ├── save-energy.php    # Сохранение энергии
│   ├── save-plus.php      # Сохранение улучшений
│   ├── save-skin.php      # Сохранение скина
│   └── leaderboard.php    # Таблица лидеров
└── admin/                 # Админ-панель
    └── index.php          # Страница администратора
```

## Функции

- Система авторизации (регистрация/вход)
- Игровая механика кликера
- Система улучшений
- Система скинов
- Система энергии
- Таблица лидеров
- Анти-автокликер защита

## Установка

1. Скопируйте файлы на ваш веб-сервер
2. Настройте подключение к базе данных в PHP файлах
3. Загрузите все зависимости
