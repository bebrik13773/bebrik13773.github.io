# Future Roadmap

- Вынести оставшийся тяжёлый клиентский код из `Код/index.html` в отдельные файлы: `anti-cheat-client.js`, `main-ui-client.js` и `client-log-client.js`, чтобы основной HTML перестал быть монолитом.
- Добавить плановую автоархивацию forensic-логов по retention-политике, а не только ручной запуск из админки: отдельная обслуживающая задача, лимиты по батчам и понятные статусы последнего прогона.
- Расширить maintenance screen в админке до полноценного центра обслуживания: очередь тяжёлых экспортов, прогресс длинных операций и журнал последних сервисных задач.
- Сделать отдельный forensic-viewer для обжалования банов: готовые пресеты `100ms pattern`, `multitouch`, `network error`, быстрые сводки по ритму и таймлайн подозрительных событий.
- Довести runtime-кэш админки дальше списка аккаунтов и лога пользователя, если shared hosting начнёт тормозить на карточках профиля, таблицах и служебных отчётах.
- Освежить витрину магазина ещё глубже: крупный предпросмотр выбранного улучшения, короткие анимации статусов и единый компактный паттерн для всех модальных подтверждений.
